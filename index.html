Computer Graphics

Assignment 4

Computer graphics is sub-divided into two categories:

    RASTER:
  • Raster images deal with pixels on the screen. It is used mainly for rendering images (e.g. Photographs). Images are represented by ‚Äúdots‚Äù or ‚Äúpixels‚Äù in a set resolution (number of dots/pixels per inch).
    ---
  • Raster images are characterised by becoming fuzzy as you zoom/enlarge the image. As the image is represented by dots, as you enlarge the image the dots become bigger.
    ---
  • Raster images are mainly used for photographs and image representation.
    VECTOR:
  • Vector images use mathematical functions or algorithms to display images on the screen. e.g. a line is represented by two points and an equation to draw the line between points.
    ---
  • Vector images are characterised by clean lines/edges and the ability to zoom in/out as much as you want without distortion, as each image is ‚Äúdrawn‚Äù using an algorithm.
    ---
  • Vector images are great for logos, signs, fancy text and design files.

    3D Printing & Laser Cutting

    Assignment 7

    What is 3D Printing?


    3D printing, also known as Additive Manufacturing (AM) is a term to describe set of technologies that create 3D objects by adding layer-upon-layer of material. Materials can vary from technology to technology. But there are some common features for all Addictive Manufacturing, such as usage of computer together with special 3D modeling software.

    Firstly, you will need to go https://ultimaker.com/software/ultimaker-cura to download Cura

    You can create a cad file and export to stl file. You can also go online and search for thingiverse for millions of free downloadable 3D models

    After you export the stl file, open Cura

    Next, open the stl file that you wish to print

    Click on (settings) -> (printer) Choose the exisiting printer you are using. In this, I will be using Ultimaker Cura 2+

    With that, open the custom bar and choose Material (PLA) and Nozzle size 0.4mm

    Slicing it would give you an estimated 3D printing time. Save it in your thumbdrive and now IT'S READY TO PRINT!

    What is Laser Cutting?

    Laser cutting is a process where object is placed in the X-Y plane and a LASER (Light Amplification by Stimulated Emission of Radiation) is moved across the profile to be cut. The process of laser cutting is controlled by the speed, intensity and number of pulses per second.

    Before we laser cut our object, we need to export the file format to the preferred DXF, PDF or SVG.

    Open your design in Fusion 360. In the object Browser, expand and show the bodies/components list. If you have multiple components, make sure that the root and top-most object is selected.

    Select the body which you want to extract the profile. To help you, you can turn off the bodies which are not of interest In this case only the FRONT component is selected Click onto the body surface to select the sketch plane.

    Make a new sketch on this FRONT plane:

    1. Create > Create Sketch

    2. Select(L-Click) the body surface to select the sketch plane.

    3. Important Click again on the profile (it should turn BLUE), showing you the sketch profile.

    4. Click FINISH SKETCH

    5. Your sketch should be located in the Object Browser > Sketches

    6. Select ‚ÄúFinish Sketch‚Äù

    7. Rename the sketch to identify it

    Export the profile as DXF:

    1. Right-click on the Sketch FRONT_DXF

    2. Select the Save As DXF Option

    3. Enter an appropriate name for the file e.g. FRONT

    4. Note of the Folder where the document is to be saved.

    Each laser cutter has a local PC interface with CorelDraw pre-installed as this is the default software supplied with the machine.

    CorelDraw Import DXF

    1. Create a new CorelDraw file (the default template of 40x28 inch or 1017x711 mm, the size of the laser cutter, is available).

    2. File > Import > Select DXF file

    3. In the ‚ÄúImport AutoCAD File‚Äù popup. ensure that the units are Metric and the Scaling is 1:1

    4. Place the layout on the drawing.

    Profiles are laid out to minimize material waste. You can also arrange your profiles to speed up the cutting process. Check that the profiles are correct before cutting. To cut click ‚ÄúPrint‚Äù.

    A dedicated computer system with a dedicated function within a larger system to perform a specific task.

    What is an Arduino?

    An Arduino System comprises of Software & Software Tools including : Integrated Development System (IDE), Arduino programming language (C++ like), Development & Debugging tool, Software libraries (open source contributions).

    Hardware such as ATMel processor board, Shields (Add-on modules), Sensors, actuators, peripherals.
